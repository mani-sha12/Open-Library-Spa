# 📚 Open Library Book Search App

## 🔍 Project Overview
This is a Single Page Application (SPA) built using React.js that allows users to search and browse books using the Open Library API.

---

## 🚀 Features
- Search books by title
- Display book cover, title, and author
- Responsive design (mobile, tablet, desktop)
- Loading indicator
- Error handling for API requests

---

## 🛠️ Technologies Used
- React.js
- JavaScript (ES6)
- HTML5 & CSS3
- Open Library API

---

## 🌐 API Used
Open Library Search API  
https://openlibrary.org/search.json?q=<search_term>
